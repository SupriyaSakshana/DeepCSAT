DeepCSAT

E-commerce Customer Satisfaction Score Prediction

Project Overview

DeepCSAT is a machine learning-based system designed to predict customer satisfaction scores in an e-commerce platform. It uses Artificial Neural Networks (ANN) to analyze customer behavior and order data to forecast satisfaction levels.

Technologies Used

  • Python
  • Artificial Neural Network (ANN)
  • TensorFlow / Keras
  • Data Preprocessing & Feature Engineering

Objective

The objective of this project is to help businesses identify dissatisfied customers early and improve customer experience through predictive analytics.
